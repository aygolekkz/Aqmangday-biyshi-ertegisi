Contributing to Aygolek.kz,
👍🎉 First off, thanks for taking the time to contribute! 🎉👍

The following is a set of guidelines for contributing to Aygolek and its packages, which are hosted on GitHub. 
These are mostly guidelines, not rules. Use your best judgment, and feel free to propose changes to this document in a pull request.

How Can I Contribute?

1) Reporting Typo errors
2) Paraphrasing
3_ Any suggestions to change the text, names in the story
4) Suggesting Enhancements
